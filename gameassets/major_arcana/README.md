# Celestial Major Arcana

Twenty-two individually generated Major Arcana card illustrations in a consistent celestial Art Nouveau engraving style.

The card images are numbered from `00` through `21` in traditional Major Arcana order. Each uses warm ivory paper, black engraved linework, restrained antique-gold accents, and constellation geometry intended to provide anchor points for later 2D animation.

Generation method: one built-in Create Image call per card, with a shared deck specification and a card-specific symbolic composition.
