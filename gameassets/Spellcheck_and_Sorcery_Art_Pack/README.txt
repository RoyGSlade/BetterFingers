Important transparency note
Several images were requested with transparent backgrounds, but the generated PNGs contain a visible checkerboard rendered into RGB pixels rather than true alpha transparency. The final five UI concepts were produced together as one composite sheet rather than five separate files. These files are preserved exactly as generated.

Production repairs
The checkerboard-backed icon, troll, goblin department, suggestion-box mimic,
and red-tape dragon were repaired non-destructively as matching *_v2.png files.
Their subjects were preserved while the painted checkerboard was replaced by
an intentional parchment/archive backdrop. The game serves only these v2
versions; the original generated files remain untouched for provenance.

Naming
Files are numbered in the intended production order and renamed according to their game role.
